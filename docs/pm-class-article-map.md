# PM Class Textbook Article Map

## 제외 범위

- `01~05 의사소통 기술`
- `06~12 고객 분석`

사용자 요청에 따라 위 범주는 읽기 대상에서 제외했다.

## 기존 글과의 중복 판단

기존 사이트 글은 아래 축에 이미 가까웠다.

- 고객 해석
- 기본 협업
- 제품 습관 형성
- 시스템 설계 회고
- 인터페이스 회고

따라서 새 글은 아래처럼 겹침을 피했다.

- `고객 분석`을 직접 다시 쓰지 않고 `문제 재정의`와 `지표 해석`으로 이동
- `협업 팁` 대신 `리더십 / 심리적 안전감`으로 확장
- `제품 일반론` 대신 `MVP / PRD / 로드맵 / 성장`으로 구체화
- `인터페이스 회고` 대신 `좋은 디자인 / 디자인 시스템`으로 정리

## 교재 범주를 바탕으로 만든 새 글 분류

### 문제 재정의
- 13 문제 해결 및 분석적 사고: 시스템 사고와 프로덕트
- 14 문제 해결 및 분석적 사고: 문제 재정의와 메타인지

### 팀과 리더십
- 15~22 팀워크 및 리더십 개발

### 실행
- 23~29, 33 프로젝트 관리
- 특히 제품 개발 생애 주기, MVP, PRD 이후 설계, 애자일 실행

### 데이터
- 42~51 데이터 분석 기본
- 특히 핵심 지표, 리텐션, A/B 테스트, 데이터 기반 의사결정

### 전략
- 34 시장 진출 전략과 성장 관리
- 52~59 제품 전략 및 로드맵

### 기술
- 37~41 기술 동향 및 업계 인사이트
- 특히 생성형 AI, 기술 불확실성, 기술 도입 판단

### 디자인
- 60~67 UX/UI 디자인
- 특히 좋은 디자인의 본질, 정보 구조, 디자인 시스템

## 최종 기사 세트

이 프로젝트에는 위 범주를 반영해 아래 성격의 글을 생성했다.

- 문제 재정의 2편
- 팀과 리더십 1편
- 실행 3편
- 데이터 2편
- 전략 3편
- 기술 2편
- 디자인 2편

총 15편이며, 날짜는 2025년 9월부터 2026년 3월 29일까지 교차 배치했다.
